# Wintilz

Wintilz is a Windows 11 styled quality-of-life toolkit with clean, native-feeling utilities like Caps Lock Display, Typing Clicks, and a Dropover-style drag shelf/tray.

## Features
- **Caps Lock Display** compact overlay with size, style, and position controls
- **Typing Clicks** configurable key sounds with profiles, volume, and repeat handling
- **Dropover** floating shelf/tray for quick file staging via drag-and-drop
- **Windows 11-style UI** modern settings layout and theming

## Settings Storage
Settings are saved to:
```
%LocalAppData%\Wintilz\settings.json
```
Dropover temporary files are stored in:
```
%LocalAppData%\Wintilz\Dropover
```
